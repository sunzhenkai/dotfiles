# WezTerm 快捷键

> **Leader 键：`Ctrl + a`**（先按 Leader，再按对应键）
> **滚动备选：`Cmd + ↑/↓`**（macOS 原生滚动，始终可用）

## 滚动

| 快捷键 | 功能 |
|--------|------|
| `Leader + [` | 向上滚动半页 |
| `Leader + ]` | 向下滚动半页 |
| `Cmd + ↑` | 向上滚动一行 |
| `Cmd + ↓` | 向下滚动一行 |

## 复制模式

| 快捷键 | 功能 |
|--------|------|
| `Leader + v` | 进入复制模式（再按 `Ctrl+v` 可矩形选择） |
| `Escape` | 退出复制模式 |

## 窗格 (Pane)

| 快捷键 | 功能 |
|--------|------|
| `Leader + \` | 水平分割（左右） |
| `Leader + -` | 垂直分割（上下） |
| `Leader + h` | 切换到左边窗格 |
| `Leader + l` | 切换到右边窗格 |
| `Leader + k` | 切换到上边窗格 |
| `Leader + j` | 切换到下边窗格 |

## 调整窗格大小（`Leader + r` 进入）

> 进入后可持续按键调整，按 `Escape` 退出。

| 快捷键 | 功能 |
|--------|------|
| `h` / `←` | 向左调整 |
| `l` / `→` | 向右调整 |
| `k` / `↑` | 向上调整 |
| `j` / `↓` | 向下调整 |
| `Escape` | 退出调整模式 |

## 窗格快速切换（`Leader + p` 进入）

> 1 秒无操作自动退出。

| 快捷键 | 功能 |
|--------|------|
| `h` / `←` | 切换到左边窗格 |
| `l` / `→` | 切换到右边窗格 |
| `k` / `↑` | 切换到上边窗格 |
| `j` / `↓` | 切换到下边窗格 |

## 其他

| 快捷键 | 功能 |
|--------|------|
| `Leader + c` | 打开命令面板 |
