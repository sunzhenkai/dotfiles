# iConfig

```shell
# install
git clone git@github.com:sunzhenkai/iconfig.git ~/.config/iconfig
cd ~/.config/iconfig && make all

# config
cd ~/.config/iconfig/init
./activate

# config oh-my-zsh
vim ~/.zshrc
ZSH_CUSTOME=~/.config/zsh/oh-my-zsh

# install programs
ii homebrew -i
ii homebrew -c

ii asdf -i
ii asdf -c

ii zsh -c
```
