# Ghostty 快捷键

> **Leader 序列：`Ctrl + A`**（按 `Ctrl+A`，松开，再按对应键）
> **重载配置：`Cmd + Shift + ,`**

## 滚动

| 快捷键 | 功能 |
|--------|------|
| `Leader + [` | 向上滚动一页 |
| `Leader + ]` | 向下滚动一页 |
| `Leader + ↑` | 滚动到顶部 |
| `Leader + ↓` | 滚动到底部 |

## 选择与复制

| 快捷键 | 功能 |
|--------|------|
| `Leader + V` | 进入选择模式 |
| `Cmd + C` | 复制选中内容 |
| `Cmd + V` | 粘贴 |
| `Cmd + F` | 搜索 |

## 窗格 (Split)

| 快捷键 | 功能 |
|--------|------|
| `Leader + \` | 水平分割（左右） |
| `Leader + -` | 垂直分割（上下） |
| `Leader + H` | 切换到左边窗格 |
| `Leader + L` | 切换到右边窗格 |
| `Leader + K` | 切换到上边窗格 |
| `Leader + J` | 切换到下边窗格 |
| `Leader + =` | 均分所有窗格 |

## 窗格调整大小（`Leader + R` 进入）

> 每次按 `Leader > R > 方向键` 调整 10 像素。

| 快捷键 | 功能 |
|--------|------|
| `H` / `←` | 向左调整 |
| `L` / `→` | 向右调整 |
| `K` / `↑` | 向上调整 |
| `J` / `↓` | 向下调整 |
| `Escape` | 结束序列 |

## 标签页 (Tab)

| 快捷键 | 功能 |
|--------|------|
| `Leader + T` | 新建标签页 |
| `Leader + N` | 下一个标签页 |
| `Leader + P` | 上一个标签页 |
| `Leader + 1~9` | 跳转到第 N 个标签页 |

## 字体大小

| 快捷键 | 功能 |
|--------|------|
| `Leader + +` | 增大字体 (+1pt) |
| `Leader + Ctrl + -` | 减小字体 (-1pt) |
| `Leader + 0` | 重置字体大小 |

## 窗口与全屏

| 快捷键 | 功能 |
|--------|------|
| `Leader + F` | 切换全屏 |
| `Leader + X` | 关闭当前 surface |
| `Leader + Q` | 关闭整个窗口 |

## 调试

| 快捷键 | 功能 |
|--------|------|
| `Leader + I` | 切换 Inspector |
